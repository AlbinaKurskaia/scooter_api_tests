Финальный проект: Автоматизация API сервиса Яндекс Самокат
Этот проект содержит автотесты для проверки функциональности создания и получения заказа в сервисе Яндекс Самокат.

Содержимое проекта
test_get_order_by_track.py: Основной файл с автотестом. Проверяет, что по треку заказа можно получить данные о нем.

sender_stand_request.py: Функции для отправки HTTP-запросов (POST и GET).

configuration.py: Хранение базового URL и путей к эндпоинтам.

data.py: Данные для запросов (тело заказа).

sql_queries.sql: SQL-запросы к базе данных (задания 1 и 2).

Скриншот запуска теста.

Инструменты и технологии
Python 3

Библиотека Pytest (для запуска тестов)

Библиотека Requests (для работы с API-запросами)

Инструкция по запуску
Установите необходимые библиотеки (если они еще не установлены):
pip install requests pytest

Запустите тест командой в терминале:
pytest test_get_order_by_track.py
